# pokemon-app
`-yarn install`<br>
`-yarn start`<br>

### Deploy heroku: 
* heroku login<br>
* heroku create <br>
* heroku apps:rename <nome_app> <br>
* git init<br>
* heroku git:remote -am "deploy"

### Install:
* yarn add serve ou npm install serve<br>

### package.json
- "start": "serve -s build"
